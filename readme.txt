Permite editar las palabras base para "Autor", "Buscar" y "Página" desde "Ajustes > Enlaces permanentes"

== Description ==

Al activar este plugin, aparecerán tres nuevos campos en el formulario de configuración de "Enlaces permanentes".
Con estos campos podrás editar las palabras base (slug) que se usan para "Author", "Search" y "Page".
Si los dejas en blanco, se usarán los valores por defecto de Wordpress.


== Installation ==

1. Descarga el plugin, descomprímelo. Aparecerá una carpeta llamada autor-buscar-pagina
2. Sube esa carpeta a la carpeta "wp-content/plugins"
3. Ve a la página de configuración de plugins de WP y actívalo
4. Ve a la sección de "Enlaces permanentes", donde podrás editar los campos "Autor", "Buscar" y "Página"


== Frequently Asked Questions ==

= He activado el plugin pero no me funciona =

Una vez activado el plugin, debes visitar al menos una vez la página de "Enlaces permanentes" y poner las palabras que quieras usar

= He cambiado las palabras base ¿debo preocuparme por el SEO? =

Por supuesto, si las páginas con la URL antigua están enlazadas o indexadas en algún buscador, deberías hacer redirecciones 301
